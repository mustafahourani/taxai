TEE
Compute Enclave
AMD SEV-SNP (verified)

This app is running inside a hardware-encrypted enclave on EigenCompute using AMD SEV-SNP. The server operator cannot read the enclave's memory. You can independently verify this deployment on-chain.

View verification on EigenCompute

RAM
Data at Rest
not stored

Nothing is saved. Your W-2 data, computed tax return, and Form 1040 exist only in the server's temporary working memory (RAM) while your request is being processed, similar to how a calculator forgets a number after you clear it. Nothing is ever written to a hard drive, a database, or a log file. Once your PDF is returned, the data is discarded. There is no account, no history, and no way for anyone to retrieve your information later.

TLS
Data in Transit
TLS encrypted

All data between your browser and the server is encrypted with TLS, the same encryption that protects your online banking. Your W-2 is scrambled before it leaves your device and unscrambled only at the destination. Normally, the server's operating system handles this unscrambling, which means the server operator could theoretically intercept it. When running inside EigenCompute, the unscrambling happens inside the hardware enclave itself, so the encrypted data passes through the server's network and operating system without ever being readable, and only becomes visible inside the protected enclave where your taxes are computed.
